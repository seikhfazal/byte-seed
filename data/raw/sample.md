# ByteSeed Sample Notes

Data structures help programmers organize information. An array stores items in order. A linked list stores items in nodes, where each node points to the next node.

A stack is a linear data structure that follows last in, first out. Programmers use stacks for undo systems, parsing, and function calls. A queue follows first in, first out and is useful for scheduling work.

Artificial intelligence is the study of systems that can perform tasks that seem intelligent. A language model learns patterns in text and predicts the next token from previous tokens.

Programming is the practice of writing precise instructions for a computer. Clear names, small functions, and tests make programs easier to understand.

Machine learning models improve by seeing examples. A tiny model trained on a tiny dataset can learn simple phrases, but it cannot understand the world like a large production model.
